How To Join

For Java Edition

  1. Launch Minecraft: Java Edition.
  2. Click on "Multiplayer," then "Add Server."
  3. Enter play.slime-smp.com in the "Server Address" field.
  4. Click "Done" and double-click the server to join!

For Bedrock Edition

  1. Launch Minecraft on your device (Windows 10, Console, Mobile).
  2. Click "Play," then navigate to the "Servers" tab.
  3. Scroll down and click "Add Server."
  4. Server Name: SlimeSMP
  5. Server Address: play.slime-smp.com
  6. Port: 19132
  7. Click "Save" and join the server!
